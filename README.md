# shoppingSystem
> 信息系统作业
系统的用户可能来自世界各地，如果依然固执的认为时间都是北京时间，就有些固步自封了。时区的问题非常复杂，不时会让人陷入迷惘之中。一旦考虑到多时区的问题，这一切立即变得复杂起来。<br><br>
本系统能够支持跨时区分布，无论用户身处何方，订单的信息都能显示本地时间
